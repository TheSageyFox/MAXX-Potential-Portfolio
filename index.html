Personal Profile

Hi, I'm Keeley, the creator of The Code Fox website and thank you for visiting my profile page. Within the confines of this site you'll find some of the projects that I've worked on while learning front-end development. I hope that you find within this site idea of the sort of code I am capable of writing. Designing and coding websites is one of my passions, I enjoy creating something beautiful, functional, and unique from mere words.

I love to work with small businesses and non-profits to allow them to have a professional and easy-to-use home on the web for their potential clients, customers, or anything inbetween. Not only can I code websites from scratch, but I also have the ability to write SEO targeted content and have done so in the past.

I've been working on and off with web design since I was in high school, before it became common practice to seperate content and design. I've been writing not only website content for friends, but also engaging in creative writing for even longer than that.

I realize that I don't seem to have a vast background or fancy college degree to me name, but what I do have is a dedication to excellence, desire to succeed, and appetence for knowledge. I strive to work to the very best of my capability and beyond, because of my disabilities. I endeavor to overcome my limitations and push myself to become someone that my friends and family can be proud of.

If you're interested in my skills to create a website of your very own, or would like to chat with me about an employment opportunity please feel free to reach out. I would like to be very transparent in the fact that I cannot speak over the phone, and all of my contact is through email, text or face-to-face meetings. Once again, thanks for checking out my profile.

Front-end Development Projects

Memory Match

My first project written in JavaScript and jQuery. The most difficult aspect of this project was learning how to map the array of cards to produce the deck programatically rather than writing coding the deck in HTML. This project was created as part of the nanodegree program through Udacity. My favorite aspect of this project was the feeling of complete acomplishment after getting the desk of cards to load and shuffle properly.

FEND Frogger

My second major JavaScript/jQuery project and by far my most difficult. The hardest part of the coding for this project was linked to the positioning of the sprite and removing the ability for the sprite to travel out of the playing area.My favorite aspect of this project was figuring out how to tie the movement of the sprite into the arrow key input on the keyboard.

MoonRaven Rabbitry

A small, responsive website designed for MoonRaven Rabbitry. Built using a mobile-first approach. The most challenging part of this build was understanding what aspects of Bootstrap to use for the Photo Gallery page. My favorite aspect of this project was the fact that I had complete control over the design and code since I was creating the site for my own rabbitry. When I created the site I had to rely on an outside source for the contact page.

St. John's UCC

A mobile landing page design for St. John's UCC located in Richmond, Virginia. A project I challenged myself to create in order to explore a clean, minimalistic design that was easy to use for an elderly congregation when accessed on a mobile device. This was one of my favorite projects when it comes to simplistic design, I was really happy with the use of whitespace.
